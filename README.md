# tunableOCLbenchmarks
This is a tunable extension for PolyBench/ACC OpenCL benchmarks. The exetension is based on the PolyBench/ACC OpenCL benchmarks which are developped in University of Delaware https://github.com/cavazos-lab/PolyBench-ACC by Scott Grauer-Gray, William Killian, John Cavazos, Robert Searles and Lifan Xu. The extension is straight forward by merging the original work groups in different granulity to change the number of work group in the working space.
